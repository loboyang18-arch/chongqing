# 重庆电力现货市场电价预测 — 实验报告（精简版）

> **阅读说明**：本文件为汇报用精简版。完整技术细节见 **`实验报告.md`**。运行产物在 **`output/`**（默认不提交 Git）；超参 JSON 在 **`params/`**。

---

## 一、核心结论（请先读本节）

### 1. 我们最终主张什么

| 主张 | 说明 |
|------|------|
| **不要用 MAE 单指标选 DA 主模型** | MAE 最低的模型往往把 24 小时曲线压成“平线”，**形状几乎不可用**；对报价/套利/偏差管理，**日内曲线形状**往往比再抠 1～2 元 MAE 更重要。 |
| **DA 推荐主线：Level + Shape（当前落地为 V5 Profile）** | 在 **MAE 可接受**（与调参 LGB 同量级）的前提下，**曲线轮廓**（峰谷走势、方向、振幅）显著优于纯 LGB。 |
| **RT 要拆成两件事** | **正常结构**可用 Level+Shape；**尖峰/近零价等异常**单靠回归学不稳，需要 **风险识别 + 分流/回退**，否则 MAE 与形状会互相拉扯。 |
| **继续涨分主要靠数据与特征，不靠换更复杂模型** | 有效训练约 **99 天**；深度学习、Transformer 在此数据量下 **未超过** 树模型 + 简单形状先验。 |

### 2. 数字结论（测试集，与工程输出一致）

**DA 日前**

| 对比项 | 调参 LGB（MAE 导向） | **V5 Profile（推荐综合方案）** |
|--------|----------------------|--------------------------------|
| MAE（元/MWh） | **17.05**（最低） | 17.27（约 +1.3%） |
| 日曲线相关 profile_corr | 0.22（形状差） | **0.51**（约 **+130%**） |
| 方向准确率 direction_acc | 0.24 | **0.58**（约 **+140%**） |
| 日内振幅误差 amplitude_err | ~59 | **~54**（**更敢起伏**） |
| 综合分 Composite（越低越好） | 0.52 | **0.41**（更贴近“水平+形状”业务目标） |

**RT 实时**

| 目标 | 更合适的参考模型 | 说明 |
|------|------------------|------|
| **尽量压低 MAE** | 加权集成等，约 **MAE 29** | 形状指标仍弱，偏“点误差最优”。 |
| **压低振幅误差、兼顾结构** | **V5 Profile**，amplitude_err 约 **79**（全实验对比中更优一档） | MAE 会升高；RT 需在报表里 **同时看 MAE 与形状/振幅**。 |

### 3. 三条最重要的“失败教训”（避免重复踩坑）

1. **只优化 MAE → 模型学会“缩到均值附近”**，图上像平线，峰谷错位、振幅不足。  
2. **99 天样本 + 强重叠序列 → Transformer/复杂序列模型泛化差**，不如 **近日形状惯性（如 3 日均值）** 稳。  
3. **RT 的 spike** 若当普通回归一起做，会 **拖垮形状** 或 **牺牲 MAE**；应 **规则/风险时段分流**，不追求单点精准预测尖峰。

## 二、业务上怎么用本报告

- **DA**：日常研判以 **V5 Profile**（或同架构 Level+Shape）为主输出；需要对标“监管/报表式 MAE”时，可同时列 **调参 LGB**，但应注明 **形状指标差异**。  
- **RT**：决策若依赖 **极端时段**，勿只看点预测；建议 **正常模型 + 高风险时段告警/回退**（V5 中已实现一版规则 + fallback）。  
- **选模**：已引入 **Composite Score**（DA/RT 权重不同），见 `output/v5_profile/all_models_ranked.csv`。

---

## 三、数据与任务（极简）

- **数据**：重庆现货相关公开数据，**27** 个源文件 → 清洗聚合为小时宽表 → **DA 75 维 / RT 81 维** 特征（含爬坡、状态 flag、形状先验、RT-DA 价差等）。  
- **泄漏控制**：按可得时间 **lag0 / lag1(24h) / lag2(48h)** 分层；滚动分位数等用 **expanding**，避免用未来信息。  
- **切分**：训练 **2025-11-01 ~ 2026-02-08（99 天）**；测试 **2026-02-09 ~ 2026-03-10（30 天）**。  
- **难点**：最强单特征与电价相关约 **0.37**；**无 r>0.5 特征**；测试期 RT 波动 **明显大于** 训练期（分布偏移）。

## 四、方法演进（一条线看清版本）

| 阶段 | 名字 | 一句话 |
|------|------|--------|
| 早期 | 基线 LGB、调参、集成、CNN-LSTM 等 | 在 **MAE** 上打磨；发现 **曲线太平**。 |
| V1 | Level + Shape 混合 | 用 **日均价 + 日内偏离** 拼曲线；**形状相关**大幅好于纯 LGB。 |
| V2 | 自适应 Level | LGB 与 **EMA/朴素日均价** 等混合，**日均价 MAE** 下降（约 11% 量级）。 |
| V3 | 增量实验 | ramp/flag、分位数、样本加权、spread 目标等；**部分特征长期并入主线**。 |
| V4 | Shape 评估 + Huber | 建立 **6 个形状指标**；DA 用 **Huber** 减轻“压平”；**统一横评**历史结果。 |
| **V5** | **Profile Forecast** | **日级模型**管 Level；**日振幅模型 + 缩放**管起伏；RT **spike 规则 + 回退**；**Composite** 选模。 |

**不必记细节时**：只需记 **V5 = 当前推荐的“profile 导向”完整工程版**。

## 六、后续最值得做的三件事（按优先级）

1. **拉长历史 + 滚动重训 Level**：改善 **趋势转折期**（如持续低价周）的日均价锚定。  
2. **接入更接近实时的偏差类特征**（负荷/新能源实际偏差等）：针对 **RT 波动与 spike**。  
3. **维持双指标汇报**：至少同时给 **MAE/RMSE** 与 **profile_corr / direction_acc / amplitude_err**，避免再次“唯 MAE”。

## 附录 A：形状指标含义（查阅用）

| 指标 | 含义 |
|------|------|
| profile_corr | 按日算 24 点预测与实际的 **Pearson 相关**，再对天平均 |
| norm_profile_mae | 按日 **z-score** 后比形状，弱化绝对水平 |
| peak / valley hour err | 预测与实际的 **峰/谷所在小时** 偏差 |
| amplitude_err | 日内 **max−min** 与实际的差距 |
| direction_acc | 相邻小时 **涨跌方向** 一致比例 |

实现见 `src/shape_metrics.py`；批量评估见 `src/evaluate_shapes.py`。
